Leveraging the open source Partio library for easily reading/writing/manipulating common particle formats, Partio4Maya extends the functionality of particles in maya with a set of Maya nodes and commands implemented as a C++ plugin. These include:

An openGL visualization tool for loading massive particle sets into the Maya UI.
A functional cache emitter that reads particle data from any partio format into a particle or Nparticle Object.
A drop in replacement for a particle object in a particle instancer workflow. Supports rotation/scale/instanceID.
An Intuitive GUI for exporting particle data, wrapped around a drop in replacement for the dynExport maya command.
A Maya command to import a single particle cache into a maya particle node as an initial state.

Partio4Maya currently supports: PDA,PDB,PDC,MC,PRT,GEO,BGEO,BIN,PTC,RIB and lidar/pointCloud formats: PTS,XYZ,PCD

(not all formats are both read/write)
